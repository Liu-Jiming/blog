Hexo + Github + Netlify 博客部署指北

写在开始之前

笔者曾六次尝试过使用 Hexo 搭建个人博客，在这六次尝试中，遇到过不少报错，也曾苦于没有全面、细致的教程来辅助搭建，走了不少弯路。于是，我便有了写一篇细致、易懂、方便排错的 Hexo 教程的想法。

这篇教程耗时三个小时写成，希望能够帮到你。

这篇教程是在 Windows 11 系统下写成的，使用其他版本 Windows 或者 Mac 的同学需要根据自身情况灵活使用。

Tips

  • 建议遵照这篇教程中的步骤，以免发现问题时为时太晚，只能从头来过。
  • 不要修改 hosts。盲目修改 hosts 很可能进一步拖慢访问速度。如果访问实在太慢，建议珂学上网。
  • 不要把 Hexo 文件和生成的页面放到同一个分支。共用分支会使后续同步更麻烦。
  • 保持耐心，只要沉住气，耐心部署，你一定会成功！

准备工作

0.1 安装 NVM 并安装 Node.js

你可以通过这个链接下载 NVM : https://hub.fastgit.xyz/coreybutler/nvm-windows/releases

在 Assets 菜单下，下载 nvm-setup.exe。

运行 nvm-setup.exe，安装 NVM 。

下载完成后，右键任务栏中的 Windows 图标，在菜单中选择 「 Windows 终端（管理员）」，打开终端。

Windows 10 没有自带 「 Windows 终端 」，可以使用 PowerShell 。
